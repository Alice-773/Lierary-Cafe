# ☕ Literary Café

> Um sistema web para gerenciamento de acervo literário desenvolvido com Django.
O **Literary Café** nasceu de uma necessidade pessoal. Como leitora, percebi que, com o passar do tempo, ficava cada vez mais difícil controlar quais livros eu já possuía, quais estavam disponíveis e evitar compras repetidas.
O projeto começou como uma atividade acadêmica para desenvolver um CRUD em Django, mas evoluiu para um MVP que une desenvolvimento web, experiência do usuário e identidade visual inspirada em cafeterias literárias e clubes de leitura.

---

## ✨ Funcionalidades

- 🔐 Sistema de autenticação (Login e Logout)
- 📚 Cadastro de livros
- ✏️ Edição de registros
- 🗑️ Exclusão de livros
- 📊 Dashboard com estatísticas do acervo
- 📖 Controle de disponibilidade dos livros
- ⚙️ Painel administrativo utilizando Django Admin
- 🎨 Interface personalizada inspirada em cafeterias literárias

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/literary-cafe.git
```

Entre na pasta:

```bash
cd literary-cafe
```

Crie um ambiente virtual:

```bash
python -m venv venv
```

Ative o ambiente virtual.

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute as migrations:

```bash
python manage.py migrate
```

Crie um superusuário:

```bash
python manage.py createsuperuser
```

Inicie o servidor:

```bash
python manage.py runserver
```

Acesse:

```
http://127.0.0.1:8000/
```
